    Worksheet Review 11- Substitution and Area
    

    Instructions: Prepare neatly written solutions to the following problems. Show all relevant work and use proper notation/terminology. Calculators should NOT be used.
    🔗

    1. Substitution Method.

    Evaluate \(\displaystyle \int 2x \sqrt{1+x^2} \, dx\) using substitution.
    🔗
    Solution.
    Let \(u = 1 + x^2\text{,}\) then \(du = 2x \, dx\text{.}\) Substituting into the integral gives:
    \begin{align*} \int 2x \sqrt{1+x^2} \, dx \amp = \int \sqrt{u} \, du\\ \amp = \frac{2}{3} u^{3/2} + C\amp = \frac{2}{3} (1+x^2)^{3/2} + C \end{align*}
    🔗
    🔗
    🔗

    2. Integration by Substitution.

    Evaluate \(\displaystyle \int x \sin(x^2) \, dx\)
    🔗
    Solution.
    Let \(u = x^2\text{.}\) Then \(du = 2x \, dx\text{,}\) which implies \(x \, dx = \frac{1}{2} du\text{.}\) Substituting these into the integral:
    \begin{align*} \int x \sin(x^2) \, dx \amp = \int \sin(u) \cdot \frac{1}{2} \, du\\ \amp = \frac{1}{2} \int \sin(u) \, du\\ \amp = -\frac{1}{2} \cos(u) + C \end{align*}
    Substituting back \(u = x^2\text{:}\)
    \begin{equation*} -\frac{1}{2} \cos(x^2) + C \end{equation*}
    🔗
    🔗
    🔗

    3. Area Between Curves.

    Find the area of the region enclosed by the graphs of \(y = x^2 - 2\) and \(y = x\text{.}\)
    🔗
    Solution.
    First, find the points of intersection by setting the equations equal to each other:
    \begin{equation*} x^2 - 2 = x \implies x^2 - x - 2 = 0 \implies (x - 2)(x + 1) = 0 \end{equation*}
    The curves intersect at \(x = -1\) and \(x = 2\text{.}\)
    🔗
    On the interval \([-1, 2]\text{,}\) the line \(y = x\) is above the parabola \(y = x^2 - 2\text{.}\) The area \(A\) is given by:
    \begin{align*} A \amp = \int_{-1}^{2} [x - (x^2 - 2)] \, dx = \int_{-1}^{2} (-x^2 + x + 2) \, dx\\ \amp = \left[ -\frac{x^3}{3} + \frac{x^2}{2} + 2x \right]_{-1}^{2}\\ \amp = \left( -\frac{8}{3} + \frac{4}{2} + 4 \right) - \left( \frac{1}{3} + \frac{1}{2} - 2 \right)\\ \amp = \left( -\frac{8}{3} + 6 \right) - \left( \frac{1}{3} - \frac{3}{2} \right)\\ \amp = \frac{10}{3} - \left( -\frac{7}{6} \right) = \frac{20}{6} + \frac{7}{6} = \frac{27}{6} = 4.5 \end{align*}
    The area of the region is \(4.5\) square units.
