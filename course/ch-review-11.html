    Worksheet Review 09- Antiderivatives, Newton’s Method and Linearization
    

    Instructions: Prepare neatly written solutions to the following problems. Show all relevant work and use proper notation/terminology. Calculators should NOT be used.
    🔗

    1. Indefinite Trigonometric Integration.

    Evaluate the following integral:
    \begin{equation*} \int \left( \sin(2x) + \cos\left(\frac{x}{2}\right) \right) \, dx \end{equation*}
    🔗
    Solution.
    Using the integration rules for trigonometric functions with linear arguments:
    \begin{align*} \int \sin(2x) \, dx + \int \cos\left(\frac{x}{2}\right) \, dx \amp = -\frac{1}{2}\cos(2x) + \frac{1}{1/2}\sin\left(\frac{x}{2}\right) + C\\ \amp = -\frac{1}{2}\cos(2x) + 2\sin\left(\frac{x}{2}\right) + C \end{align*}
    🔗
    🔗
    🔗

    2. Integration of Exponential Functions.

    Evaluate \(\displaystyle \int (e^x - e^{-x})^2 \, dx\)
    🔗
    Solution.
    First, expand the integrand:
    \begin{equation*} (e^x - e^{-x})^2 = e^{2x} - 2e^x e^{-x} + e^{-2x} = e^{2x} - 2 + e^{-2x} \end{equation*}
    Now, integrate term by term:
    \begin{align*} \int (e^{2x} - 2 + e^{-2x}) \, dx \amp = \frac{1}{2}e^{2x} - 2x - \frac{1}{2}e^{-2x} + C \end{align*}
    🔗
    🔗
    🔗

    3. General Antiderivatives.

    Find the general antiderivative of \(f(x) = -5x^5 + \frac{1}{x^2} - 3\sqrt{x}\text{.}\)
    🔗
    Solution.
    Rewrite the function in power form: \(f(x) = -5x^5 + x^{-2} - 3x^{1/2}\text{.}\) Apply the power rule for integration:
    \begin{align*} F(x) \amp = -5\left(\frac{x^6}{6}\right) + \frac{x^{-1}}{-1} - 3\left(\frac{x^{3/2}}{3/2}\right) + C\\ \amp = -\frac{5}{6}x^6 - \frac{1}{x} - 2x\sqrt{x} + C \end{align*}
    🔗
    🔗
    🔗

    4. Linearization and Approximation.

    Find the linearization of \(f(x) = \sqrt{x+3}\) at \(a=1\) and use it to approximate \(\sqrt{3.98}\text{.}\)
    🔗
    Solution.
    The linearization is \(L(x) = f(a) + f'(a)(x-a)\text{.}\)
    1. \(\displaystyle f(1) = \sqrt{1+3} = 2\)
      🔗
      🔗
    2. \(\displaystyle f'(x) = \frac{1}{2\sqrt{x+3}} \implies f'(1) = \frac{1}{2(2)} = \frac{1}{4}\)
      🔗
      🔗
    Thus, \(L(x) = 2 + 0.25(x-1)\text{.}\)
    🔗
    To approximate \(\sqrt{3.98}\text{,}\) note that \(x+3 = 3.98 \implies x = 0.98\text{:}\)
    \begin{equation*} f(0.98) \approx L(0.98) = 2 + 0.25(0.98 - 1) = 2 + 0.25(-0.02) = 2 - 0.005 = 1.995 \end{equation*}
    🔗
    🔗
    🔗

    5. Newton’s Method.

    Starting with \(x_1 = 2\text{,}\) find the third approximation \(x_3\) to the root of \(x^3 - 2x - 5 = 0\text{.}\)
    🔗
    Solution.
    Let \(f(x) = x^3 - 2x - 5\text{,}\) so \(f'(x) = 3x^2 - 2\text{.}\) The formula is \(x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\text{.}\)
    🔗
    First iteration (\(n=1\)):
    \begin{equation*} x_2 = 2 - \frac{f(2)}{f'(2)} = 2 - \frac{8-4-5}{12-2} = 2 - \frac{-1}{10} = 2.1 \end{equation*}
    Second iteration (\(n=2\)):
    \begin{equation*} x_3 = 2.1 - \frac{f(2.1)}{f'(2.1)} = 2.1 - \frac{(2.1)^3 - 2(2.1) - 5}{3(2.1)^2 - 2} \approx 2.1 - \frac{0.061}{11.23} \approx 2.09457 \end{equation*}
    🔗
    🔗
